Sticky

Usage

<div data-sticky-container> // The container
  ...
  <div data-sticky-element>Element to be stickied inside the container</div>
  ...
</div>

Options

on container

Option type default description
data-sticky-container-ignore-padding-top boolean true ...
data-sticky-container-ignore-padding-bottom boolean false ...
data-sticky-throttle number 50 ...

on element

Option type default description
data-sticky-element-ignore-margin-top boolean false ...
data-sticky-element-ignore-margin-bottom boolean false ...
data-sticky-element-ignore-border-top boolean true ...
data-sticky-element-ignore-border-bottom boolean false ...
data-sticky-element-z-index number 1 ...
data-sticky-element-min-window-width number 0 Stick the element only if the window width is greater than this value.

on other elements

Option type default description
data-sticky-external-top-element - - ...
data-sticky-external-bottom-element - - ...
data-sticky-stack - - Split the container in 2 or more stacks, each sticking elements independently, useful for bootstrap cols.
